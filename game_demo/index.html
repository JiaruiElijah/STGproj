        怪物编辑器

        此处为全局最高优先级：敌人生命、攻击、移速、金币、检测范围、碰撞半径等均由本面板决定，并保存到本地；战斗中生成敌人时只读此处，波次配置不再覆盖数值。STG 纵版射击另可编辑每种怪的弹幕类型（瞄准玩家 / 直线向下 / 随机 / 无弹幕）、弹幕冷却与敌弹速度。

        波次阵型编辑器

        种类与数量仅由下方格子决定。三块棋盘为主战场上/左/右延伸；左键按住拖动可连续摆放；右键擦除一格。同一格多只怪可写 type1|type2（需分次填入或导入）。「出兵间隔 / 灵力 / 下一波」为全局节奏参数；出怪顺序与格子遍历顺序一致（同类型合并为连续组）。旧存档仅有种类+数量时，会尝试摆到上棋盘以便编辑。
