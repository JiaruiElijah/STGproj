敌弹贴图编辑器

与怪物编辑器共用本地存档 tower_defense_enemy_types。此处只改敌弹（敌人子弹）位图，不影响自机子弹。每种怪填写 art_assets/bullets/ 下的文件名；选文件只会写入文件名，你必须把同名文件放到 game_demo/art_assets/bullets/（或上一级 STGproj/art_assets/bullets/）。点「应用并保存」后游戏会带版本参数重新加载，避免浏览器仍显示旧图。清除贴图或留空并应用后，该种类敌弹仅矢量。

章节设置

一局游戏按顺序通关各章；每章有独立的波次与「升级时刻」。增加章节时在末尾追加空章（进入波次阵型可编辑）；减少章节会删除末尾章节且不可恢复。

怪物编辑器

此处为全局最高优先级：敌人生命、攻击、移速、金币、检测范围、碰撞半径等均由本面板决定，并保存到本地；战斗中生成敌人时只读此处，波次配置不再覆盖数值。STG 纵版射击另可编辑每种怪的弹幕类型（瞄准玩家 / 直线向下 / 随机 / 无弹幕）、弹幕冷却与敌弹速度。下方每种怪默认折叠，点击标题行展开。

波次阵型编辑器

种类与数量仅由下方格子决定。先选章节再编辑该章内的波次；多块棋盘为主战场上/左/右延伸；左键按住拖动可连续摆放；右键擦除一格。同一格多只怪可写 type1|type2（需分次填入或导入）。用「在此波前/后插入」可在任意两波之间加入新波并自动选中新波编辑；选最后一波再点「之后插入」即末尾追加。「出兵间隔 / 灵力 / 下一波 / 本波血量倍率」随当前波次变化；出怪顺序与格子遍历顺序一致（同类型合并为连续组）。旧存档仅有种类+数量时，会尝试摆到上棋盘以便编辑。

怪物血量倍率（按当前波次）

以怪物编辑器中该种类的血量为基准，在下方「当前波次」一行中设置本波敌人血量倍率（每一波可单独填写）。全局随波次自动增长公式已暂时禁用；旧档里若仍含「基础/线性/加速」系数，仅用于首次打开面板时把未写倍率的波次换算成数值，保存后即写入各波的 enemyHpMult。

敌弹停火判定（全局 · 按行）

主战场棋盘从上往下数第几行（与 STG 棋盘行一致）。敌人中心 Y 到达或超过该行下沿后，不再发射战斗弹幕；未进入棋盘前也不发射。死后弹幕不受影响。留空=不启用。旧存档若仅有像素线，打开面板时会按约 45px/行估算成行号。

升级时刻（局内四选一强化）

在当前章节内第 N 波结束后（进入下一波前）插入升级时刻。此前因经验累计升了几级，就会连续进行几轮「四选一」。多个波次用逗号分隔，如 1,3,5。留空并保存表示本章不在波次衔接插入升级时刻；若该章从未写过此项，游戏内仍默认在第 1 波结束后插入一次。
